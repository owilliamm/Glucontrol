<h2>Introdução</h2>
Glucontrol é um pequeno sistema capaz de ajudar no gerenciamento de informações importantes para o diabético. Tem como função principal auxiliar no controle glicêmico, apresentando histórico essencial para que o usuário possa repensar o dia a dia a partir da detecção de momentos com constante hipoglicemia e/ou hiperglicemia.


<h2>Função</h2>
O programa permite criar, editar, buscar e remover registros com facilidade, via um menu intuitivo por terminal, guardando as seguintes informações para que possam ser acessadas a qualquer momento:
<pre>
 ‧ Aferições de glicemia
 ‧ Carboidratos consumidos
 ‧ Insulina aplicada
 ‧ Data e horário
 ‧ Minutos de prática aeróbica e de resistência
</pre>
